# 🌸 Petal Kitchen - My Personal Recipe Hub

Dobrodošli u **Petal Kitchen**, moju web aplikaciju za recepte! 🍳✨ 
Ovaj projekat sam napravila kako bih na jednom mjestu imala inspiraciju za kuhanje, planiranje obroka i pametnu shopping listu.

## 🚀 Šta sve Petal Kitchen može?

* **Nasumična inspiracija:** Dugme "Inspire Me" generiše nove ideje za ručak svaki put kad ti zatrebaju. 🪄
* **Pametno filtriranje:** Traži recepte po tipu (Doručak, Ručak, Večera) ili posebnim dijetama (Veggie, Halal, No Gluten). 🥗
* **Sedmični planer:** Automatsko generisanje menija za cijelu sedmicu sa savršenim vizuelnim prikazom. 📅
* **Moja kuhinja:** * Sistem za prijavu korisnika (Petal ID). 🔑
    * Čuvanje omiljenih recepata (Favorites). ❤️
    * Shopping lista sastojaka koja se čuva u bazi podataka. 🛒

## 🛠️ Tehnologije koje sam koristila

* **Frontend:** HTML5, CSS3 (Custom Grid & Flexbox), JavaScript (ES6+).
* **Backend:** Node.js & Express.
* **Baza podataka:** SQLite.
* **API:** [Spoonacular API](https://spoonacular.com/food-api) za povlačenje hiljada recepata.
* **Ikonice:** FontAwesome.

## 📸 Kako pokrenuti projekat lokalno?

1.  Kloniraj repozitorij: `git clone https://github.com/sary0-0/Coolinarka.git`
2.  Instaliraj zavisnosti: `npm install`
3.  Pokreni server: `node server.js`
4.  Otvori `index.html` u pretraživaču.
